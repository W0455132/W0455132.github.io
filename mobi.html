Mobile Mini-Proects

A collection of projects completed over the course of my third term at NSCC. Each project was meant to expose us to a different unique property of developing in Android Studio.

DINDER

Restaurant matcher

Application for helping a group of users narrow down a list of restaurants created through user-defined parameters. Final self-directed project for NSCC.


Demonstrates:

  • Google Maps and Places API
  • Firebase user authentication tools
  • Serverless real-time database interactions
  • Full application design and architecture

TO-DO APP

Task Organization App

This application was a part of a group defined and directed project in our Project Management course.


Demonstrates:

  • SQLite data storage
  • CRUD methods
  • Intent creation and usage

MAPS

Place picker

Introduction to Google Maps. This project was built from an empty project using Android Maps documentation to integrate Google Maps. It allows the user to search for locations to add Markers and Circles to.


Demonstrates:

  • API creation and secure storage
  • Accessing device location
  • Managing user permissions
  • Map Markers and Circles

WEB SERVICES

Chuck Norris Joke Generator

This application interacts with the REST API at http://www.icndb.com/api/ . It retrieves a random 'Chuck Norris Joke' from the web service and displays it on the screen.


Demonstrates:

  • REST API interaction
  • Random library

DATABASE CONNECTION

Muppet Database

This application was a part of a group project in our Project Management course. The project was self-defined and self-directed.


Demonstrates:

  • SQLite data storage
  • CRUD methods
  • Intent creation and usage

INTENTS

Image Markup

A simple app that allows the user to draw on a blank screen or select an image from their camera roll to draw on.


Demonstrates:

  • Bitmap interaction
  • Intent access
  • User permissions
